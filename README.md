# node-indonesia-subdivision-area

**WORK IN PROGRESS**

Pustaka node.js untuk list daerah-daerah administratif Indonesia.

Data meliputi nama dan kode:
* provinsi
* kabupaten/kota
* kecamatan
* kelurahan
